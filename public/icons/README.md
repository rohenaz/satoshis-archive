# Icon Files

This directory should contain the application icons for the desktop app.

## Required Files

- `icon.png` - 512x512 PNG for Linux
- `icon.icns` - macOS icon bundle
- `icon.ico` - Windows icon file
- `icon-template.png` - 16x16 or 22x22 PNG for system tray (transparent background)

## Generate Icons

You can use tools like:
- [electron-icon-builder](https://www.npmjs.com/package/electron-icon-builder)
- [electron-builder icon guide](https://www.electron.build/icons)
- Online icon generators
